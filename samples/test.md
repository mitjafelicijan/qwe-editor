# Learn Markdown

Markdown is a lightweight markup language for formatting plain text. Itâ€™s widely used for README files, documentation, notes, and writing on platforms like GitHub.

## 1. Headings

Use `#` characters for headings. More `#` means a smaller heading.

```md
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

---

## 2. Paragraphs & Line Breaks

Just write text normally.

* Leave a **blank line** between paragraphs.
* Add two spaces at the end of a line for a manual line break.

---

## 3. Emphasis

```md
*Italic* or _Italic_
**Bold** or __Bold__
***Bold and Italic***
~~Strikethrough~~
```

*Italic*
**Bold**
***Bold and Italic***
~~Strikethrough~~

---

## 4. Lists

### Unordered Lists

```md
- Item
- Item
  - Nested item
* Item
```

### Ordered Lists

```md
1. First
2. Second
3. Third
```

## 12. Escaping Characters

Use a backslash (`\`) to escape Markdown characters.

```md
\*This is not italic\*
```

*This is not italic*
